# INTAN Studio - Modern Fashion E-Commerce Website

## Project Overview

INTAN Studio is a modern fashion e-commerce website developed as the Final Project (UAS) for the Computer Application IT course at International Women University (IWU).

The purpose of this project is to demonstrate front-end web development skills by creating an interactive online fashion store using HTML, CSS, JavaScript, and Bootstrap 5. The website provides a responsive and user-friendly shopping experience for customers.

---

# Business Overview

## Business Name

INTAN Studio

## Business Description

INTAN Studio is an online fashion store that offers modern clothing for young adults. The website provides a simple and attractive shopping experience where users can browse products, search items, add products to the shopping cart, and simulate online payments.

## Value Proposition

- Modern and trendy fashion products
- Responsive website design
- Easy navigation
- Simple shopping experience
- Affordable product prices

## Target Market

The target customers are:

- College students
- Young adults (18–30 years old)
- Fashion lovers
- Online shoppers

## Competitor Analysis

INTAN Studio competes with several online fashion brands such as:

- Uniqlo
- H&M
- Local fashion stores
- Shopee Fashion Store
- Tokopedia Fashion Store

The advantage of INTAN Studio is its clean interface, responsive design, and simple shopping flow.

## Product Management Strategy

The products displayed on the website include various fashion items complete with:

- Product image
- Product name
- Product price
- Product description
- Size selection

Products are displayed in a clean catalog layout to make browsing easier.

## Business Model

INTAN Studio uses a Business to Customer (B2C) business model where customers purchase products directly through the website.

## Pricing Strategy

The products are offered at affordable prices suitable for students and young adults. Promotional discounts can be added in future development.

## Checkout & Payment

The website includes a checkout page and payment simulation. After completing payment, users are redirected to the Payment Success page.

## SEO Strategy

Basic SEO implementation includes:

- Semantic HTML structure
- Descriptive page titles
- Responsive design
- Organized navigation

## Security & Maintenance

- Shopping cart data is stored using Local Storage.
- Website files are organized into separate folders.
- Product data and website content can be updated easily.

## Future Analytics Plan

Future improvements may include:

- Google Analytics integration
- Visitor statistics
- Product popularity analysis
- Customer behavior tracking

# Technical Explanation

This project is built using HTML5, CSS3, Vanilla JavaScript, and Bootstrap 5.

The website layout is designed using Bootstrap's responsive grid system, allowing the website to work properly on desktop and mobile devices.

The project is organized into separate folders for better maintainability:

- **assets/** contains website resources such as the logo, hero banner, and product images.
- **css/style.css** contains all website styling and responsive layouts.
- **js/script.js** manages the main website interactions.
- **js/products.js** displays product data and handles product search.
- **js/product-detail.js** displays detailed information about each product.
- **js/cart.js** manages shopping cart operations such as adding and removing products.
- **js/cart-badge.js** updates the cart icon badge.
- **js/payment.js** manages the payment simulation process.

Shopping cart information is stored using Local Storage, allowing users to keep their selected products even after refreshing the browser.
