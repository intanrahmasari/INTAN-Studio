# INTAN Studio - Modern Fashion E-Commerce Website

INTAN Studio is a modern fashion e-commerce website built using HTML, CSS, Bootstrap 5, and JavaScript. This project was developed as a final assignment (UAS) for the Komputer Aplikasi IT course to demonstrate front-end web development skills by creating an interactive online fashion store.

---

## Features

- Home Page
- Product Catalog
- Product Search
- Category Filter
- Product Detail Page
- Size Selection
- Add to Cart
- Shopping Cart
- Remove Product from Cart
- Cart Badge
- Local Storage
- Payment Page
- Payment Success Page
- About Page
- Responsive Design

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla JS)
- Bootstrap Icons
- Google Fonts
